# Gplay-Redeemer
Automated Tool That Redeems discord trails from gsuite accounts 





# How To use ? 

- Download Ldplayer9 Set Your instance to 1920x1080 (tablet) , download discord 309.11 From Apkmirror , Enabled adb and root in ldplayer settings
- open platform tools folder , copy the frida file and paste in diskC/users/LDplayer9/ (where u have installed the ldplayer paste there ) 
- now go and run `run.py` several times untill the frida server says all completed 
- then run server.py (if localhost directly run it , if runing on rdp , copy the api url of rdp and replace it in your api.txt ) 
- put ur materials and run card adder
